## Key Accomplishments

### Phase 15 ‚Äî Quality & Platform

1. **EF Core 10 RTM Upgrade** ‚Äî All packages already at `10.0.9` stable. No code changes needed.
2. **CSV Streaming for Large Imports** ‚Äî `ReadCsvStreamAsync()` + `ImportFromCsvStreamAsync()` added with `IAsyncEnumerable`, lazy row processing, and batch save every 50 rows. ImportViewModel splits into two paths at 1,000-row threshold. Progress reporting via `IProgress<int>`.
3. **Activity Feed Pruning** ‚Äî `AccessLogCleanupService` with configurable retention (default 30 days, 0 = disabled). `AccessLogCleanupHostedService` runs on startup + daily via `PeriodicTimer` in BrokerService. Also runs on standalone startup in CatalogBrowser.
4. **Re-scan Folder Wiring** ‚Äî `FolderScanService` with full pipeline: discovery ‚Üí validation ‚Üí checksum ‚Üí duplicate check ‚Üí thumbnail ‚Üí metadata extraction ‚Üí ingest. `SidebarViewModel.RescanFolderAsync()` replaces TODO with actual scan call.
5. **Test Coverage Expansion** ‚Äî 34 deferred tests created across 5 areas: SearchService (9), AccessLogCleanupService (5), FolderScanService (3), CSV streaming (8), PropertyInspectorViewModel batch guards (9). All tests pass.

### Phase 16 ‚Äî Provenance & Trust

1. **Data Model** ‚Äî `IsAiGenerated` property added to `Keyword` and `Category` models with default `false`.
2. **Service Layer** ‚Äî `isAiGenerated` parameter on `AssociateKeywordsAsync()`/`AssociateCategoriesAsync()`; `AiTaggingService` passes `true` for AI-generated tags.
3. **Wire Protocol** ‚Äî `KeywordInfo`/`CategoryInfo` field 5, `AssetDetail` field 27 for `TagsAreAiGenerated`.
4. **Broker Handlers** ‚Äî `SidebarHandler` populates provenance; `AssetHandler.UpdateAssetAsync()` preserves `IsAiGenerated` when re-associating keywords.
5. **ViewModel Tracking** ‚Äî `_aiGeneratedTags` in `MetadataEditorViewModel`; `AiGeneratedTagNames` in `PropertyInspectorViewModel`.
6. **UI Badge** ‚Äî `TagOccurrence.IsAiGenerated` model; `TagEditorControl.AiGeneratedTagNames` dependency property; AI badge template with tooltip.
7. **EF Core Migration** ‚Äî `AddIsAiGenerated` migration with Designer file. All 1,097 tests pass.
